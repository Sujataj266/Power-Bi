# SuperStore-Sales-Analysis (Interactive Dashboard creation using Power BI)

## Overview
Analyze SuperStore sales performance by tracking sales, profit, orders, shipping modes, customer segments, and regional trends. The project provides actionable business insights through an interactive Power BI dashboard with KPI metrics and visual reports.

## Dataset Used
- <a href="https://github.com/Sujataj266/Power-Bi/blob/main/SuperStore%20Sales%20Data/SuperStore_Sales_Dataset.csv">SuperStore-Sales-Dataset</a>

## KPIs
- Goal: Analyze and improve business sales performance.
- Guidelines:
1. Use retail sales datasets.
2. Identify sales and profit trends across regions, categories, and segments.
3. Analyze customer behavior and shipping performance.
4. Build an interactive dashboard with business KPIs.
