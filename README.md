# Crop Yield Prediction Dashboard ((Interactive Dashboard creation using Power Bi))

## Project Overview

This project focuses on Crop Yield Prediction and Analysis using Power BI, aiming to support data-driven decision-making in agriculture. The dashboard analyzes historical agricultural data to identify yield patterns, risk factors, and sustainability insights that can help farmers, planners, and policymakers improve productivity.
